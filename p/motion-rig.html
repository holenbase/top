五款AE绑定脚本对比评测

5个MG人物动画绑定脚本

在AE软件中有很多用于角色绑定的工具，这里对比了Duik | Rubberhose | Limber | Character Tools | Puppet Tools这5个脚本。



Duik Bassel

版本：v16.2.28
Duik是一款免费开源的动力学和动画工具。动画的基本工具包含：反向动力学、骨骼变形器、动态效果、自动骨骼绑定，IK，图形学等。它提供了可在任何3D软件（如IK，控制器和骨骼）中找到的主要装配工具，但可将它们适应After Effects中的2D动画。Duik还拥有许多动画工具 -管理关键帧和插值，传统动画，动画曝光等-以及诸如摆动，弹簧，摆动，滚轮等自动化功能，可让您更快，更轻松地进行动画处理。
【点击下载】 【官方下载】



Rubberhose

版本：v2.0.7
RubberHose是一款制作MG人物绑定动画的AE脚本，是专为处理动画部分的开放系统，将更好的控制角色的性能。仅需两个控制点即可完全控制整个绑定动画，不同物体间的链接动，可自由变形拉伸，利用脚本能够快速生成人体的手和脚，然后还可以和身体做绑定，制作人体动画。
【点击下载】



Limber

版本：v1.6.0
Limber是一个用于After Effects的IK系统，它使用形状图层为角色动画制作可定制的肢体。Limber有控制样式的形状和颜色，动态防止动画“弹出”，顺利地融合之间的IK和FK，并交换四肢的不同类型。
【点击下载】



Character Tools

版本：v1.0.6
Character Tool 是由Motion Design School开发的制作人物角色绑定动画的多功能工具。它包含装配工具，工作流程增强器和许多辅助运动预设。
【点击下载】



Puppet Tools

版本：v3.7.5
Puppet Tools通过Null空物体来控制卡通人物角色骨骼，相当于3D软件的IK，十分方便，对于做角色Motion动画对话，非常便利。
【点击下载】



——比较：

Duik的功能最丰富也最强大，而且它是一个开源免费的脚本，但是这也导致绑定费时，对于那些希望把精力更多花在动画上的人不是一件好事。 Rubberhose最适合做橡皮管风格人物，简单且出效果，缺点就是局限性大，需要额外的更多控制与变化的话比较难；
Limber和Rubberhose整体感觉差不多，比较大的缺点是不好自定义外观，但是它有一个独一无二的功能就是制作透视变化
Character Tools和Rubberhose一样可以快速绑定四肢和制作动画。
